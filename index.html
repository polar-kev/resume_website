Roll Ball: A Unity Game

Play a web version of the game here. (Mobile Not Supported)


This was my first Unity game project. The goal was to learn how to use the editor to create a puzzle game prototype. This prototype contains 4 levels that force the player to explore and learn through experimentation. Here are the things I learned and implemented:

  • The singleton pattern--created a music controller that was instantiated at the splash screen and played music through the game. The music would loop and simply continue if the player returned to the splash screen.
  • Scripted simple player control and UI
  • Play testing research through friends and family with the help of GameBucket.io

Blasty Rocket: A 2D Unity Game

Play a web version of the game here. (Mobile Not Supported)


The goal of this personal project was to learn something new about Unity while creating an original 2D game. The spaceship, baloon and platform artwork were created by me. The background images were free assets. Here are the things I learned and implemented:

  • An alternative method to infinite scrolling backgrounds
  • Object Pooling--used for platform generation to reduce memory & cpu usage.
  • Limitless platform generation cycles for an endless game
  • Unity 2D animation creation and animation event triggers

Recreating Pong in C++

The goal of this personal project was to build my familiarity with C++ and the Simple and Fast Media Library (SFML). Check out the source code and a playable mac OSX build on my github. Things I learned and implemented:

  • SFML API and libraries
  • Drawing graphics
  • Input handling
  • Collision detection & sprite tunneling
